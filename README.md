# 💻 Mundo Dev Cyber

🌐 https://cyber.mundodev.site

Sistema de apresentação dos serviços do **Mundo Dev Cyber**, um centro de serviços digitais rápidos e acessíveis em Luanda.

---

## 🚀 Sobre o projeto

O **Mundo Dev Cyber** é um website informativo criado para divulgar os serviços do cyber, facilitando o acesso dos clientes aos preços e serviços disponíveis.

---

## 🛠️ Serviços disponíveis

- 🖨️ Impressão de documentos (preto e branco / colorido)
- 📄 Cópias e digitalização (scanner)
- 📚 Plastificação de documentos
- 📖 Encadernação de trabalhos
- 🎓 Trabalhos académicos
- 💿 Envios de músicas, filmes e séries
- 🎮 Jogos
- 💻 Instalação de sistemas operativos
- 🧾 Informatização de documentos
- 📚 Venda de material didático

---

## ⚡ Tecnologias usadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Bootstrap Icons
- AOS (animações)

---

## 🎯 Objetivo

Oferecer uma plataforma simples, rápida e acessível para clientes visualizarem os serviços e preços do cyber.
